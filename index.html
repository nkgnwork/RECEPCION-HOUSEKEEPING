Ve a supabase.com → Start your project → regístrate con email (sin tarjeta)
Paso 2
Crea proyecto → región South America (São Paulo) → espera ~2 min
Paso 3
Project Settings → API → copia la Project URL y la anon public key
Paso 4 — Una sola vez en SQL Editor
